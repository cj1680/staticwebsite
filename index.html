Struggles with this assignment

I had many difficulties with this assignment, the first being setting up the git repository in a format that can be deployed by AWS without error. I kept getting an error because I did not include the source files directly, but rather had the directory files for both the boilerplate website and the bootstrap assets. Once I placed just index.html and the css/js assets, it went very smoothly.
